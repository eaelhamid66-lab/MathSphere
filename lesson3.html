Linear Functions

Explanation

A linear function represents a constant rate of change and forms a straight line when graphed. It is written in the form f(x)=mx+b, where m represents the slope and b represents the y-intercept. The slope shows how fast the output changes relative to the input. Linear functions are widely used to model relationships such as speed, cost, and growth with constant change. Understanding slope direction, intercepts, and graph behavior allows students to interpret real-life data effectively.

Examples

  • f(x)=2x+3 → slope=2
  • f(x)=-x+5 → decreasing line
  • If x increases by 1, y changes by m
