Quadratic Function

Explanation

A quadratic function is a polynomial function of degree two written in the standard form f(x)=ax²+bx+c where a≠0. The graph of a quadratic function is called a parabola and it opens upward if a is positive and downward if a is negative. Quadratic functions are used to model curved motion such as projectile paths and optimization problems. Important features include the vertex, axis of symmetry, intercepts, and maximum or minimum value. Students learn how coefficients affect the shape and position of the graph. Completing the square and factoring are common techniques used to analyze quadratic equations.

Examples

  • f(x)=x² → parabola opening upward
  • f(x)=-x²+4 → maximum value at vertex
  • Vertex of x²-4x+3 is (2,-1)
