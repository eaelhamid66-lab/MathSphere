Real Functions

Explanation

A real function is a mathematical relation where every real input value corresponds to exactly one real output value. Functions describe how one quantity depends on another and are essential in algebra, physics, and engineering. The main idea is consistency: an input cannot produce two different outputs. Functions are usually written using function notation such as f(x). Understanding functions allows students to analyze patterns, model real-life situations, and predict behavior of systems mathematically. The graph of a function visually represents how inputs transform into outputs.

Examples

  • f(x)=x² → f(3)=9
  • f(x)=2x+1 → f(4)=9
  • Each x has only ONE value of f(x)
