Domain and Range

Explanation

The domain of a function represents all possible input values that can be substituted into the function without causing mathematical errors such as division by zero or square roots of negative numbers. The range represents all possible outputs produced by those inputs. Determining domain and range helps understand where a function exists and what results it can generate. Students often analyze restrictions caused by fractions, radicals, or logarithmic expressions. Graphically, the domain extends along the x-axis, while the range extends along the y-axis.

Examples

  • f(x)=1/x → domain: x ≠ 0
  • f(x)=√x → domain: x ≥ 0
  • Range of x² → y ≥ 0
